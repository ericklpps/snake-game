Como Jogar

  • O jogador deve controlar a cobra com as setas
  • A cobra deve comer as maçãs no cenário
  • A cada maçã comida, a cobra cresce
  • Se a cobra colidir com a parede ou com seu próprio corpo, o jogo termina
  • O objetivo é comer o máximo de maçãs possíveis sem colidir
  • O jogo utiliza os comandos de setas no teclado, podendo se mover para cima, para baixo e para ambos os lados
  • Se mover no próprio eixo é proibido e pode causar sua derrota
  • Ao final, é permitido reiniciar diretamente ou adicionar sua pontuação no ranking
  • Ao iniciar, o jogador pode escolher entre diferentes níveis de dificuldade, sendo fraco o padrão
  • A cada nova rodada, a dificuldade retorna para o padrão, caso queira um desafio diferente, altere antes de iniciar
